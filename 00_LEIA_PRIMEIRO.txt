quick_scenario_test.py
├─ Mostra como ficaria Q26 em dois cenários
├─ CENÁRIO 1: line_height=14px → Q26 em Y=365px
├─ CENÁRIO 2: line_height=18px → Q26 em Y=465px
├─ Diferença acumulada: +100px
└─ Saída: Visual clara do impacto

diagnose_grid_alignment.py
├─ Lê o arquivo JSON (block_*_coordinates_adjustment.json)
├─ Calcula o ideal_line_height (17.58px → 18px)
├─ Verifica se ajuste será ativado
├─ Mostra diferença acumulada por questão
└─ Saída: Análise teórica completa

measure_real_spacing.py
├─ Carrega imagens reais de blocos
├─ Detecta bolhas com cv2.HoughCircles
├─ Agrupa por linha (questão)
├─ Mede espaçamento vertical (média, desvio padrão)
├─ Compara com 14px e 18px
└─ Saída: Estatísticas de espaçamento real

generate_grid_overlay.py
├─ Carrega imagens de blocos
├─ Desenha grid VERDE (14px) com transparência
├─ Desenha grid VERMELHO (18px) com transparência
├─ Sobrepõe na imagem original
├─ Salva em debug_corrections/grid_overlay/
└─ Saída: Imagens para comparação visual

compare_expected_vs_real.py
├─ Carrega todas as configurações de blocos
├─ Calcula posições esperadas
├─ Mostra tabela comparativa
├─ Detecta qual valor está sendo usado
├─ Fornece plano de ação específico
└─ Saída: Recomendação + instruções
